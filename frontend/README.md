# Order Management Frontend

SPA Vue3 para visualização de pedidos em tempo real.

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

A aplicação estará disponível em `http://localhost:3000`

## Build

```bash
npm run build
```

## Funcionalidades

- ✅ Dashboard de pedidos
- ✅ Filtros por status e cliente
- ✅ Atualizações em tempo real via SignalR
- ✅ Autenticação JWT
- ✅ Suporte a multitenancy
